Existing zone file records in excess of 100 will not be deleted. Adding records to any zone over 100 will require an upgrade to Premium DNS featuring unlimited records.
A (Host)
A records (also known as host records) simply point a domain name or sub domain to an IP address.
Start A record wizard »
CNAME (Alias)
CNAME records (also known as alias records) point a subdomain to an already existing A (host) record so that the subdomain resolves to the appropriate IP address.
Start CNAME record wizard »
MX (Mail Exchanger)
MX records are for routing email that is addressed to a particular domain name. Like a CNAME record, an MX record points one domain name or subdomain to another domain name or subdomain for which an A record exists.
Start MX record wizard »
TXT (Text)
TXT records consist of a text string that communicates protocol-specific information to certain protocols such as SPF.
Start TXT record wizard »
SRV (Service)
SRV records (also known as service records) are responsible for localizing specific domain-related services such as FTP, HTTP, and SIP.
Start SRV record wizard »
AAAA (IPv6 Host)
AAAA records store the IPv6 address for a given domain name or subdomain. They follow the standard colon delimited format.
Start AAAA record wizard »
NS (Nameserver)
NS records specify the authoritative nameservers for the given domain. In every DNS zone, there must be at least two NS records.
Start NS record wizard »
SOA (Start of Authority)
The Start of Authority (SOA) record is the first zone file entry and includes authoritative information about a domains DNS zone.
